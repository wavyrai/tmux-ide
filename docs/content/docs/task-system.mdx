# Task system

This page is historical/future-facing. There is no current task/mission CLI or
orchestrator runtime implementation to present as an available workflow.
Mission runtime wiring for `.tmux-ide/workspace.yml` is future work.

tmux-ide is, first, a cockpit around tmux: adopt a session, know your fleet, and
survive crashes. Current coordinated work should be modeled in your own project
or agent prompts. WorkspaceConfigV1 can declare `harnesses`, `agents`, and
`missions` data, but runtime adapters and dispatch behavior are not wired yet.
